# Browser Development Tools

These scripts are designed to be **pasted into the browser console** for debugging and testing.
They require a browser DOM environment and cannot be run with Node.js.

## Files

| Script | Purpose |
|--------|---------|
| `selector_finder.js` | Interactive tool for finding element selectors by clicking |
| `overlay_test.js` | Test the background mode overlay UI |
| `full_agent_test.js` | Full agent test harness for console |
| `auto_accept_test.js` | Test auto-accept button detection |
| `antigravity_background_poll_test.js` | Test Antigravity background polling |
| `cursor_background_poll_test.js` | Test Cursor background polling |

## Usage

1. Open IDE (Cursor/Antigravity) with DevTools (`F12` or `Ctrl+Shift+I`)
2. Navigate to Console tab
3. Paste the entire content of the script
4. Follow script-specific instructions in console output
